# Hypertext Narrative: My Move to Cambridge

## Overview
This project is a hypertext narrative about my move from Lancaster, Pennsylvania to Cambridge, England. The story is split across multiple HTML pages and allows users to navigate through different stages of the journey.

## Pages
- index.html → Lancaster
- page2.html → Wales
- page3.html → Transition
- page4.html → Journey to Cambridge
- page5.html → Arriving in Cambridge
